# 🗺️ DevFlow-AI – Roadmap

This document outlines the **phased development plan** for this platform.

---

## ✅ Phase 1: Frontend Booking Platform Prototype

- [x] Initial React + TypeScript + Vite project setup
- [x] Responsive layout with Tailwind CSS
- [x] Interactive homepage with property filters and search queries
- [x] Property detail page (image gallery, amenities list, availability calendar)
- [x] Dynamic pricing and invoice calculations
- [x] Custom Auth modal simulating Guest & Host logins
- [x] Host Dashboard showing earnings analytics, listing management, and reviews
- [x] Static pages (AirCover, Help Centre, Terms, Privacy Policy)
- [x] Global state provider with client-side mock database

---

## 🚀 Phase 2: Full-Stack Integration & Database

- [ ] Node.js + Express backend API setup
- [ ] MongoDB database connection and schema implementation
- [ ] Actual JWT authentication with Secure Cookies & Refresh Tokens
- [ ] User role-based middleware (Guest, Host, Admin)
- [ ] Real-time guest-host messaging system utilizing WebSockets
- [ ] Image upload service integration (e.g. Cloudinary)
- [ ] Stripe & Razorpay payment gateway integration

---

## 🌟 Phase 3: AI-Powered Workspace Features

- [ ] **AI Property Description Generator**: Automate listing summaries based on features
- [ ] **AI Price Recommendation Engine**: Evaluate pricing by comparing nearby locations and occupancy trends
- [ ] **AI Review Summarizer**: Parse and digest user feedback trends automatically
- [ ] **AI Fraud Detection**: Detect fake profiles or anomalous bookings
- [ ] Complete developer assistant utilities (AI Code Review, README generator, AI error scanner)
